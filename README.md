### Regras da aplicação

[] Deve ser posssível criar um usuário.
[] Deve ser possivel identificar o usuário entra as requisições
[] Deve ser possivel atualizar os dados do usuário

[] Deve ser possivel manter um carrinho
[] Deve ser possivel obter a quatidade de itens no carrinho
[] Deve ser obter todos os produtos

[] Deve ser possivel registra o pagamento.
[] Deve ser possivel obter os pagamentos de um usuário.
